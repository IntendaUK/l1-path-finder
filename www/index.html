Theory
Instead of searching a grid cell-by-cell, l1-path-finder uses preprocessing to extract all corner points and build a visibility graph. Steiner vertices are added to reduce the size of this graph from O(n2) to O(n √ log(n)). Querying the visibility graph is done using A* with landmarks. Left click to place blocks, right click to set start/goal.
Purple lines are the visibility graph, yellow vertices are landmarks, teal vertices are corners
